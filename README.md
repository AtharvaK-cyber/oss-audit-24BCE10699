# oss-audit-24BCE10699
Open Source Audit project on Git including Linux analysis and shell scripting tasks.

Name : Atharva Katare.  

Reg no. 24BCE10699.

Software of Choice:Git. 

This project examines Git, an open-source program, and shows how to use it in a Linux environment. In order to comprehend automation and system interaction, it also includes shell scripting tasks.

1. script1.sh: Shows OS, kernel, user, and uptime information.  
2. script2.sh: Verifies whether Git is installed and displays package information  
3. script3.sh: Verifies directories and displays their size and permissions  
4. script4.sh: This program counts the instances of keywords in log files.  
5. script5.sh: This script uses user input to create an open source manifesto.# How to Run

Chmod +x script_name.sh
